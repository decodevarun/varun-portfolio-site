# Barun Pal Portfolio Website

A modern, dark-and-white portfolio website for **Barun Pal**, a **Performance Marketer & SEO Specialist** focused on **eCommerce and fashion brands**.

## Pages
- Home (`index.html`)
- About (`about.html`)
- Services (`services.html`)
- Portfolio (`portfolio.html`)
- Contact (`contact.html`)

## Features
- Sharp, conversion-focused copy and structure
- Clean minimal interface with dark UI + white accent
- Fully responsive navigation and layout
- Hero section with strong call-to-action
- Dedicated services for SEO, Performance Marketing, and Content Strategy
- Portfolio section with case-study cards
- Contact form with name, email, and message fields
- SEO-friendly semantic HTML and optimized static assets for speed

## Run Locally
Use any static server.

### Option 1: Python
```bash
python3 -m http.server 8080 --bind 127.0.0.1
```
Open: `http://127.0.0.1:8080`

### Option 2: VS Code Live Server
- Install the **Live Server** extension
- Right-click `index.html`
- Click **Open with Live Server**

## Performance Notes
- Lightweight static HTML/CSS/JS (no framework overhead)
- Minimal JavaScript with deferred loading
- No large media dependencies by default
